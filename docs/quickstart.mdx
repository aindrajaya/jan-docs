# Quickstart

This guide will help you to use the desired AI models, setting up the threads, and chat with the model.

<div class="text--center"> 
    <img src={installImageURL} width={600} alt="jan-ai-download" />
</div>

### Step 1: Installing Jan

Go to [jan.ai](https://jan.ai/), select your desktop operating system (**MacOS**, **Windows**, or **Linux**), and install the program. For details, go to [Installation guide](https://visionary-cajeta-bf2527.netlify.app/docs/install).

### Step 2: Select AI Model

Jan offers many AI models. Each model has their own purposes, capabilities, and different requirements. You can choose which model can fulfill your requirements based on their proficiency in the Hub. For more info, go to [list of supported models](/docs/models-list/). 

### Step 3: Use the AI Model

Now that you are fully setup, you can use the AI under the Chat screen.
